## Архитектурный бэклог (вне REQ-таблицы)

Задачи из CHANGELOG.md и AGENTS.md, не привязанные к конкретному REQ:

1. **FlashList для ChatScreen** — вместо ScrollView для сообщений. Сейчас ScrollView без виртуализации (как в веб-версии), но для больших чатов может потребоваться `FlashList` с `maintainVisibleContentPosition` + `inverted`.
2. **MMKV миграция** вместо AsyncStorage (быстрое хранилище, уже установлен `react-native-mmkv`).
3. **Profile Pager** — табы профиля на `react-native-pager-view` (уже установлен).
4. **Push-уведомления** — `expo-notifications` + FCM для новых сообщений при закрытом приложении.
5. **Фоновые звонки** — `react-native-callkeep` для входящих звонков при закрытом приложении.
6. **BL-1: Миграция медиа на Firebase Storage** — фото/аудио/видео как URL вместо base64 в RTDB (лимит 10 МБ).
