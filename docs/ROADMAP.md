## ✅ Блок А — Производительность чата (выполнено v4.17.0, 2026-05-25)

**Проблема пользователя:**
- Чат скролился рывками при большом количестве сообщений
- После удаления пузыря верхний чат «падал как кирпич» — layout-анимация дергалась
- Ручное отслеживание клавиатуры (~25 строк в ChatScreen) давало артефакты: инпут подпрыгивал, emoji-панель и клавиатура конфликтовали

**Решение (реализовано):**
1. ✅ **FlashList для ChatScreen** — `FlatList` заменён на `FlashList` из `@shopify/flash-list`. Убраны `removeClippedSubviews`, `initialNumToRender`, `windowSize`, `maxToRenderPerBatch`, `scrollEventThrottle`.
2. ✅ **KeyboardAvoidingView** — ручное отслеживание клавиатуры (`Keyboard.addListener` + `bottomPad`) полностью удалено. Вместо этого `KeyboardAvoidingView` (behavior="padding") из `react-native-keyboard-controller`. `bottomPad` оставлен только для emoji-панели.

**Критерий готовности:**
- Скролл чата плавный без фризов на 500+ сообщениях
- Удаление пузыря — соседние сообщения пересчитываются без рывков
- Открытие/закрытие клавиатуры — инпут движется синхронно с нативной анимацией
- Emoji-панель корректно сдвигает контент и закрывается при тапе в поле ввода

**EAS dev-build:** требуется пересборка APK (FlashList уже в dependencies, но ChatScreen теперь использует его нативный код).

## Следующие направления (приоритет по убыванию)

### Блок B — Push-уведомления (критично для мессенджера)
- `expo-notifications` + FCM для новых сообщений при закрытом/свёрнутом приложении
- Быстрый ответ из уведомления
- Настройка Firebase Cloud Messaging в консоли

### Блок C — UX-полировка (Telegram-style)
- Глобальный поиск по чатам и сообщениям
- Превью ссылок в чате
- Умные даты («вчера», «понедельник») в списке чатов
- Меню вложений (скрепка → галерея, камера, файл, контакт)
- Свайп на чате в списке (архив/закреп/удалить)
- Плавающая дата при скролле сообщений

### Архитектурный бэклог
- **BL-1: Миграция медиа на Firebase Storage** — вместо base64 в RTDB (лимит 10 МБ)
- **MMKV** вместо AsyncStorage
- **Живые стикеры через WebView** — SMIL не рендерится в react-native-svg на Android
- **Profile Pager** — табы профиля на react-native-pager-view
